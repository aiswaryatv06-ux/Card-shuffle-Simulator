# Card Shuffle Simulator (Python)

# Project Description
This is a simple Python program that simulates a standard 52-card deck.  
The program creates a deck of cards, displays the cards before shuffling, shuffles them randomly, and then displays the shuffled deck.

This project demonstrates basic Python concepts like lists, loops, functions, and the random module.


# Features
- Creates a standard 52-card deck
- Displays cards before shuffling
- Randomly shuffles the deck
- Displays shuffled cards
- Simple and beginner-friendly code


# Technologies Used
- Python
- random module
